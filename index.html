Kanban

Environment: HTML, CSS, Javascript, Vue.js, Node.js, MongoDB

This app is a recreation of a Kanban board; which allows users to create boards about specific projects or issues. Each board can have multiple lists that can each contain specific tasks that need to be completed. Each task can have their own comments. Think of Trello as an example. This project utilizs Express Sessions for authentication and bcrypt for password hashing. I worked on this project with a partner, and gained expereince in paired programming. If you think the design is unique, that is what we were going for. If you don't like it, it was all my partner's idea...

Keepr

Environment: HTML, CSS, JavaScript, Vue.js, C#, .Net Core, MySQL

This app is essentially a recreation of Pinterest. A user can view, create, and save what are called 'keeps' (posts), and create and view their saved 'vaults' (collections of posts). In order to create a keep or vault the app user must be logged in, and this project implements Auth0 for user management. It also utilizes SQL to communicate with the MySQL database. Tests were developed for the server, which gave me experience in Test Driven Development. To deploy this application on Heroku I utilized Docker containers.
